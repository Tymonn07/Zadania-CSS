Wewnętrzny arkusz stylów:

  1. gdzie jest wstawiany

    2. Wstawia się zawsze w części nagłówkowej dokumentu (pomiędzy znacznikami < head> a ). Można go zastosować, gdy elementy które pragniemy poddać formatowaniu, występują wielokrotnie na stronie i wszystkim chcemy nadać takie same atrybuty (inne niż domyślne).

  2. składnia teoretyczna,

    3. < head>
         (...)
      < style type="text/css">
      < ![CDATA[ */
          selektor1 { cecha: wartość; cecha2: wartość2... }
          selektor2 { cecha: wartość; cecha2: wartość2... }
         (...)
       /* ]]>
      
       (...)
    < /head>

  3. jakie mogą być selektory

    4. Jako "cecha" (w deklaracji stylu - powyżej) należy wpisać o jakie konkretnie atrybuty formatowania nam chodzi (opisane w kolejnych rozdziałach).

  4. jakie mogą być cechy i wartości

    5. Natomiast jako wyraz "wartość" wpisujemy dokładną wartość atrybutu. Zwróć uwagę, że jednemu selektorowi możemy nadać kilka atrybutów (cech). Są one wtedy rozdzielone średnikami.
